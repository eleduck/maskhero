# 项目配置

1. 下载代码并进入目录

   ```shell
   git clone git@github.com:eleduck/maskhero.git
   cd maskhero
   ```

2. 安装 NPM 包

   ```shell
   yarn install // 或 npm install
   ```

3. 设置环境变量

   ```shell
   cp .env.example .env.local
   ```

   或 @管理员 要正式环境变量文件

4. 启动项目

   ```
   yarn start // 或 npm run start
   ```

   访问 http://localhost:3000/ 即可访问网站

5. 构建生产环境版本

   ```
   yarn build // 或 npm run build
   ```
